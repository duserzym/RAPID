# RAPID
This repository contains the code for the RAPID paleomagnetic magnetometer controller system. 

The original VB6 code can be found at: https://sourceforge.net/projects/paleomag/. The code is publicly available under the GNU General Public License v2.0 (GPL-2.0).

In this repository, the folder named `VB6` contains the VB6 code modified from the original source for the UC Berkeley RAPID system (Hargraves magnetometer). The system was upgraded 2024-2025 and installed at the Institute for Rock Magnetism at the University of Minnesota in 2025. 

Additional RAPID gaussmeter documentation:

- [docs/fw-bell-gaussmeter-user-guide.md](docs/fw-bell-gaussmeter-user-guide.md) covers operator setup, driver installation, DLL placement, and GUI usage.
- [docs/fw-bell-gaussmeter-developer-guide.md](docs/fw-bell-gaussmeter-developer-guide.md) covers direct driver work, helper usage, and SCPI communication outside the RAPID GUI.
