# RapidPy System Shell

Lightweight launcher that opens converted subsystem apps from one panel.

## Run

```bash
cd RapidPy/system_shell
python -m pip install -r requirements.txt
python main.py
```

## Windows Build

```bat
cd RapidPy\system_shell
build_windows.bat
```

## Purpose

- Gives operators a single entry point while migration is in progress.
- Bridges standalone subsystem apps toward final integrated automation control.
